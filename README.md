# Running Alarm Robot

## Overview

Running Alarm Robot is a multidisciplinary graduation project that combines Embedded Systems, Robotics, IoT, Mobile Development, and Computer Vision into a single intelligent platform.

The project was designed to solve a common problem faced by many people: ignoring or turning off alarms without actually waking up. Instead of remaining stationary, the alarm system is mounted on a mobile robotic platform that can move autonomously, making it more difficult for users to ignore the alarm.

The robot integrates several smart technologies including obstacle avoidance, face recognition, wireless communication, environmental monitoring, and mobile application control. The project served as a practical demonstration of how multiple engineering disciplines can be combined to build an intelligent autonomous system.

---

## Key Features

### Smart Alarm System

* Real-Time Clock (RTC) based alarm scheduling
* Autonomous movement when the alarm is triggered
* Enhanced wake-up experience through physical interaction

### Autonomous Navigation

* Obstacle detection and avoidance
* Autonomous movement using ultrasonic sensors
* Safe indoor navigation

### Face Recognition

* User identification using computer vision techniques
* Face recognition model integration
* Personalized interaction capabilities

### IoT Connectivity

* ESP8266 Wi-Fi communication
* Remote monitoring and control
* Wireless data exchange

### Mobile Application

* Smartphone-based control interface
* Alarm management and configuration
* Robot interaction through a dedicated mobile application

### Bluetooth Control

* Wireless robot control using Bluetooth
* Manual operation mode

### Safety Monitoring

* Fire detection system
* Gas leakage detection
* Environmental monitoring and alerting
